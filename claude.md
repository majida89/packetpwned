# packetpwned.com — Hugo Site

## Stack
- Hugo static site generator
- GitHub Pages (auto-deploy on push to main)
- Stealth Ops theme (GitHub dark, JetBrains Mono + Inter)

## Sections
- content/writeups/ — TryHackMe writeups
- content/fitness/  — CrossFit, running, cycling logs
- content/projects/ — Engineering projects
- content/blog/     — General posts

## Workflow
After any change: git add . && git commit -m "message" && git push

## Site params
- sport types: crossfit / running / cycling
- fitness frontmatter: score, rx (bool), pr (bool)
